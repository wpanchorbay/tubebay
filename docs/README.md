# TubeBay Documentation

This is the official documentation site for [TubeBay](https://wpanchorbay.com/plugins/tubebay) — the bridge between YouTube and WooCommerce.

## Development

This documentation is built with [VitePress](https://vitepress.dev/).

### Local Setup
1. `cd docs`
2. `npm install`
3. `npm run docs:dev`

### Deployment
Documentation is automatically built and deployed from the `main` branch.
