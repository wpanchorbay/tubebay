# Request a Feature

We want to hear from you! If you have an idea for a new feature or an improvement to TubeBay, please let us know.

## How to Submit a Request

1. Visit the [GitHub Issues page](https://github.com/wpanchorbay/tubebay/issues).
2. Click **New Issue** and select the **Feature Request** template (if available) or choose **Blank Issue**.
3. Clearly describe the feature:
   - **What problem does it solve?**
   - **How should it work?**
   - **Who would benefit from it?**

## What and What Not to Request

### ✅ What We Love:
- Ideas for new mapping strategies.
- Support for additional video platforms (Vimeo, etc.).
- Improved UI/UX for the mapping dashboard.
- New placement locations for the video player.

### ❌ What Not to Request:
- **Support Requests:** For help or bugs, use the [support forum](https://wordpress.org/support/plugin/tubebay/) instead.
- **Vague Ideas:** Try to be as specific as possible.
- **Duplicate Requests:** Search existing issues before opening a new one.

## What Happens Next?
The TubeBay team and the community will review your request. We prioritize features based on:
- **Demand:** How many other users want this?
- **Effort:** How complex is the implementation?
- **Alignment:** Does it fit the core vision of TubeBay?

Thank you for helping make TubeBay better! 🙏
